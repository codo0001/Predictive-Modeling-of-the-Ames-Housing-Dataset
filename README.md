# Predictive Modeling of the Ames Housing Dataset

### This report provides an analysis of the Ames Housing Dataset. This dataset was originally compiled by Dean De Cock from the Ames City Assessor’s Office and published in the Journal of Statistics Education in 2011 (De Cock, 2011). It contains 82 variables that include lot information, building types, quality ratings, square footage, room counts, garage capacity, basement features, and sale information. A detailed overview of the dataset is provided in Table 1. 
### The primary objective of this project is to apply hypothesis testing and statistical analysis techniques to better understand the factors that drive home sale prices. This objective was framed using three research questions. Each question was evaluated using statistical methods to test hypotheses about correlations between home features and price, neighborhood effects, and median sale price predictions using home features. 
### The statistical methods utilized in this project include multiple linear regression with F-tests and individual predictor t-tests to determine coefficient impact, ANOVA F-test for a two-model comparison, logistic regression for binary classification, and binomial Z-test to evaluate classification accuracy against random chance. A 95% confidence interval is also calculated for classification accuracy in the third research question.
